# GitHub Pages

Este repositorio contiene páginas estáticas publicadas a través de GitHub Pages.

## Acceso a las Páginas

### Opción Rápida: Generador de Enlaces

<a href="https://franrojasmay.github.io/presentaciones/path_generator.html" target="_blank">**Abre el generador →**</a>

Ingresa la ruta del archivo y copia el enlace generado automáticamente.

### Instrucciones Manuales

Para acceder a cualquier página de este repositorio, sigue estos pasos:

1. Copia la ruta del archivo HTML desde el repositorio
2. Construye la URL de GitHub Pages con la siguiente estructura:
   ```
   https://franrojasmay.github.io/[ruta-del-archivo.html]
   ```
3. Pega la URL completa en tu navegador

---

## Notas Técnicas

- GitHub Pages se habilita automáticamente cuando el repositorio es público
- Los cambios en el repositorio pueden tardar algunos segundos en reflejarse en la página publicada
- Las rutas son sensibles a mayúsculas y minúsculas
- Asegúrate de que los archivos estén en formato `.html` para ser accesibles directamente
